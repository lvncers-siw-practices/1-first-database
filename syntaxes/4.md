# SQL Syntaxes 4

## 四捨五入

```sql
SELECT
    ROUND(AVG(score), 2) AS 平均点
FROM
    exam
WHERE
    subject = "デザイン";
```

```sh
+-----------+
| 平均点     |
+-----------+
|     73.75 |
+-----------+
1 row in set (0.010 sec)
```

- `CEILLING()`: 切り上げ
- `FLOOR()`: 切り上げ

## 連結

```sql
SELECT
    CONCAT("最高 ", MAX(score), " 点") AS 最高点
FROM
    exam;
```

```sh
+-------------+
| 最高点       |
+-------------+
| 最高 92 点   |
+-------------+
1 row in set (0.003 sec)
```

---

```sql
SELECT
    CONCAT_WS("-", id, subject, score) AS 最高点
FROM
    exam;
```

## NULL変換

```sql
SELECT
    name,
    COALESCE(class, "未決定") AS class
FROM
    student;
```

```sh
+---------------+-----------+
| name          | class     |
+---------------+-----------+
| 佐藤 琢磨      | CG        |
| 大塚 愛　      | Web       |
| 藤井 隆　      | Web       |
| 福原 愛　      | CG        |
| 大黒 将志      | 未決定     |
+---------------+-----------+
5 rows in set (0.001 sec)
```

## 日付時刻関数

システム情報を取得する場合、特定テーブルの参照はいらない。
Oracleデータベースでは `FROM dual` が必要。

資料に掲載されている日付時刻関数をまとめました。

| 関数                                        | 説明                             |
| :------------------------------------------ | :------------------------------- |
| `CURRENT_DATE()`                            | 現在日付                         |
| `CURRENT_TIME()`                            | 現在時刻                         |
| `CURRENT_TIMESTAMP()`                       | 現在日付時刻                     |
|                                             |                                  |
| `YEAR(日付)`                                | 年                               |
| `MONTH(日付)`                               | 月                               |
| `DAY(日付)`                                 | 日                               |
| `HOUR(時刻)`                                | 時                               |
| `MINUTE(時刻)`                              | 分                               |
| `SECOND(時刻)`                              | 秒                               |
| `DAYOFYEAR(日付)`                           | 年の日(1\~366)                   |
| `MONTHNAME(日付)`                           | 月の名前                         |
| `DAYNAME(日付)`                             | 曜日の名前                       |
| `DATE_FORMAT(日付, フォーマット)`           | 指定したフォーマットで日付を返す |
|                                             |                                  |
| `TIMESTAMPADD(unit, interval, datetime)`    | 日付時刻の加算                   |
| `TIMESTAMPDIFF(unit, datetime1, datetime2)` | 日付時刻の減算                   |

- [TIMESTAMPDIFF()関数で使用する unit 引数一覧](https://dev.mysql.com/doc/refman/8.0/ja/date-and-time-functions.html#function_timestampadd)
- [DATE_FORMAT()関数で使用できる指定子一覧](https://dev.mysql.com/doc/refman/8.0/ja/date-and-time-functions.html#function_date-format)

```sql
SELECT CURRENT_TIMESTAMP();
```

```sh
+---------------------+
| CURRENT_TIMESTAMP() |
+---------------------+
| 2026-05-28 11:20:03 |
+---------------------+
1 row in set (0.001 sec)
```

---

```sql
SELECT DATE_FORMAT(CURRENT_TIMESTAMP(), "%Y-%m-%d (%W)") AS "CURRENT_DATE";
```

```sh
+-----------------------+
| CURRENT_DATE          |
+-----------------------+
| 2026-05-28 (Thursday) |
+-----------------------+
1 row in set (0.000 sec)
```
