# SQL Basic Syntaxes

## 四捨五入

```sql
SELECT
    ROUND(AVG(score), 2) AS 平均点
FROM
    exam
WHERE
    subject = "デザイン";
```

```sh
+-----------+
| 平均点    |
+-----------+
|     73.75 |
+-----------+
1 row in set (0.010 sec)
```

- `CEILLING()`: 切り上げ
- `FLOOR()`: 切り上げ

## 連結

```sql
SELECT
    CONCAT("最高 ", MAX(score), " 点") AS 最高点
FROM
    exam;
```

```sh
+---------------+
| 最高点         |
+---------------+
| 最高 92 点     |
+---------------+
1 row in set (0.003 sec)
```

```sql
SELECT
    CONCAT_WS("-", id, subject, score) AS 最高点
FROM
    exam;
```

## NULL変換

```sql
SELECT
    name,
    COALESCE(class, "未決定") AS class
FROM
    student;
```

```sh
+---------------+-----------+
| name          | class     |
+---------------+-----------+
| 佐藤 琢磨     | CG        |
| 大塚 愛       | Web       |
| 藤井 隆       | Web       |
| 福原 愛       | CG        |
| 大黒 将志     | 未決定    |
+---------------+-----------+
5 rows in set (0.001 sec)
```

## 日付時刻関数

システム情報を取得する場合、特定テーブルの参照はいらない。
Oracleデータベースでは `FROM dual` が必要

```sql
SELECT CURRENT_TIMESTAMP();
```

```sh
+---------------------+
| CURRENT_TIMESTAMP() |
+---------------------+
| 2026-05-28 11:20:03 |
+---------------------+
1 row in set (0.001 sec)
```

```sql
SELECT DATE_FORMAT(CURRENT_TIMESTAMP(), "%Y-%m-%d") AS "CURRENT_DATE";
```

```sql
SELECT YEAR(CURRENT_DATE());
```

```sh
mysql> SELECT DAYNAME(CURRENT_TIMESTAMP());
+------------------------------+
| DAYNAME(CURRENT_TIMESTAMP()) |
+------------------------------+
| Thursday                     |
+------------------------------+
1 row in set (0.002 sec)
```

```sh
mysql> SELECT DAYOFYEAR(CURRENT_DATE);
+-------------------------+
| DAYOFYEAR(CURRENT_DATE) |
+-------------------------+
|                     148 |
+-------------------------+
1 row in set (0.002 sec)
```

```sql
SELECT DATE_FORMAT(CURRENT_TIMESTAMP(), "%Y-%m-%d (%W)") AS "CURRENT_DATE";
```

```sh
+-----------------------+
| CURRENT_DATE          |
+-----------------------+
| 2026-05-28 (Thursday) |
+-----------------------+
1 row in set (0.000 sec)
```
