# SQL Basic Syntaxes 2

## 比較演算子

```sql
SELECT * FROM student
WHERE class = "CG"
ORDER BY id;
```

```sh
+----+---------------+------------+-------+
| id | name          | birthday   | class |
+----+---------------+------------+-------+
|  1 | 佐藤 琢磨       | 1977-01-28 | CG    |
|  4 | 福原 愛　       | 1988-11-01 | CG    |
+----+---------------+------------+-------+
2 rows in set (0.003 sec)
```

## 論理演算子

```sql
SELECT * FROM exam
WHERE subject = "PC基礎"
AND score >= 70
ORDER BY id;
```

```sh
+----+----------+-------+
| id | subject  | score |
+----+----------+-------+
|  1 | PC基礎   |    92 |
|  4 | PC基礎   |    85 |
+----+----------+-------+
2 rows in set (0.003 sec)
```

## 範囲指定

```sql
SELECT * FROM student
WHERE birthday BETWEEN "1970-01-01" AND "1979-12-31"
ORDER BY id;
```

```sh
+----+---------------+------------+-------+
| id | name          | birthday   | class |
+----+---------------+------------+-------+
|  1 | 佐藤 琢磨      | 1977-01-28 | CG    |
|  3 | 藤井 隆　      | 1972-03-10 | Web   |
+----+---------------+------------+-------+
2 rows in set (0.006 sec)
```

## 列挙

```sql
SELECT * FROM student
WHERE id IN (1, 3, 5);
```

```sh
+----+---------------+------------+-------+
| id | name          | birthday   | class |
+----+---------------+------------+-------+
|  1 | 佐藤 琢磨      | 1977-01-28 | CG    |
|  3 | 藤井 隆　      | 1972-03-10 | Web   |
|  5 | 大黒 将志      | 1980-05-04 | NULL  |

3 rows in set (0.002 sec)
```

## NULL判定

```sql
SELECT * FROM student
WHERE class IS NULL;
```

```sh

|  5 | 大黒 将志      | 1980-05-04 | NULL  |

```
