## 内部結合

```SQL
SELECT
    student.id,
    name,
    subject,
    score
FROM
    student
INNER JOIN
    exam
ON
    student.id = exam.id
ORDER BY
    student.id
;
```

```SQL
SELECT
    *
FROM
    student

    exam

    student.id = exam.id
```

- `ON` は「どの行とどの行を結合するか」を決める条件
- `ON` あり … `student.id = exam.id` が一致する行だけ結合（7行）
- `ON` なし … 直積（CROSS JOIN）。student 5人 × exam 7件 = 35行。名前と点数が別人の組み合わせになる

```sql
SELECT * FROM student, exam
WHERE student.id = exam.id;
```

この書き方でもできる。
暗黙的なCROSSJOINによる指定。

## 外部結合

### 左外部結合

```sql
SELECT
    student.id,
    name,
    subject,
    score
FROM
    student
LEFT OUTER JOIN
    exam
ON
    student.id = exam.id
ORDER BY
    student.id
;
```

### 右外部結合

`id = 5` が存在しないのは、examに含まれていないからである。

```sql
SELECT
    student.id,
    name,
    subject,
    score
FROM
    student
RIGHT OUTER JOIN
    exam
ON
    student.id = exam.id
ORDER BY
    student.id
;
```
