# Basic Syntax 6

## 内部結合

```SQL
SELECT
    student.id,
    name,
    subject,
    score
FROM
    student
INNER JOIN
    exam
ON
    student.id = exam.id
ORDER BY
    student.id
;
```

- `ON` は「どの行とどの行を結合するか」を決める条件
- `ON` あり … `student.id = exam.id` が一致する行だけ結合（7行）
- `ON` なし … 直積（CROSS JOIN）。student 5人 × exam 7件 = 35行。名前と点数が別人の組み合わせになる

`select * from student, exam where student.id = exam.id;`
この書き方でもできる。
暗黙的なCROSSJOINによる指定。

```SQL
SELECT
    c.,
    c.Name AS Country
FROM
    country AS c
LEFT JOIN
    countrylanguage AS cl
ON
    c.Code = cl.CountryCode
;
```
